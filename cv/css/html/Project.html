Code-Based Secure Lock System

In this project, I was solely responsible for designing and assembling the complete security lock circuit using the Arduino Uno, a 4x4 matrix keypad, a relay module, and a 12V solenoid electric door lock. I developed the embedded logic to validate numeric passcodes entered by the user, ensuring secure and accurate access control. I also programmed the system to display real-time status messages such as "Access Granted" or "Access Denied"—on a 16x2 serial LCD screen to enhance user interaction. By integrating all components on a breadboard and managing power through a 9V battery, I created a fully functional and compact prototype. This project helped me strengthen my skills in embedded system design, microcontroller programming, and security-based hardware development.

Audio Amplifier

In this project, I was responsible for designing and assembling the complete audio amplifier circuit using core analog components like transistors, resistors, and capacitors. I calculated appropriate resistor values and configured the components based on circuit theory and Ohm’s Law to ensure optimal performance. I integrated an audio input jack to receive signals from external devices such as mobile phones and connected the output to a speaker. I used a power supply to energize the circuit and performed multiple test runs to fine-tune the audio quality, reducing unwanted noise and distortion. This hands-on experience deepened my understanding of signal amplification, analog electronics, and practical circuit troubleshooting.

R-Tx Dinasor Game

In this project, I developed an interactive version of the RTX Dinosaur Run Game using an Arduino microcontroller and an OLED display. I programmed the core game logic in Arduino IDE, including character movement, obstacle generation, collision detection, and scoring mechanism. The system was powered by a regulated power source and tested under different conditions to ensure smooth performance and minimal lag. This project enhanced my skills in embedded programming, real-time input handling, and graphical display interfacing with microcontrollers
